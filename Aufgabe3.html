1 Aufgabe 3

1.1 Interjektionen im DWDS

Diese Webseite enthält alle Interjektionen, die in der DWDS-Lemmadatenbank verzeichnet sind. Es sollte mithilfe der bereitgestellten Rohdaten eine strukturierte Website erstellt werden.

Zunächst wurde die vollständige Lemmaliste als CSV-Datei von der DWDS-Webseite heruntergeladen. Anschließend wurde die Datei in Visual Studio Code geöffnet und mithilfe der Erweiterung „Rainbow CSV“ bearbeitet. Über eine SQL-Abfrage (RBQL) wurden ausschließlich die Einträge ausgewählt, deren Wortklasse als „Interjektion“ gekennzeichnet ist.

Die gefilterten Daten (Lemma und URL) wurden danach mit regulären Ausdrücken in eine Markdown-Liste umgewandelt. Dabei wurde jedes Lemma kursiv formatiert und mit der entsprechenden DWDS-Webadresse verlinkt. Zum Schluss wurde das Markdown-Dokument mit Pandoc in eine eigenständige HTML-Datei umgewandelt und über GitHub veröffentlicht. Die folgende Liste zeigt somit alle im DWDS verzeichneten Interjektionen in alphabetischer Reihenfolge mit direkter Verlinkung zu den jeweiligen Wörterbucheinträgen.
